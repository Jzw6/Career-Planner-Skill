## ✨ 功能亮点

- **渐进式访谈**：不是表单，是对话。AI 像真正的顾问一样逐步引导，自然挖掘你的真实情况
- **6 大维度采集**：背景现状 · 技能优势 · 兴趣热情 · 副业资产 · 价值观与约束 · 目标与恐惧
- **联网数据支撑**：自动搜索目标岗位的真实薪资、招聘需求趋势、入门路径等
- **个性化报告**：每一条建议都引用你的真实回答，不是模板套话
- **一键下载**：报告生成后自动保存为 `.md` 文件，可随时查阅

---

## 📄 报告包含内容

| 板块 | 内容说明 |
|------|---------|
| 👤 个人画像 | 综合你的背景，提炼核心优势与"超能力" |
| 📊 技能盘点 | 硬技能 / 软技能 / 待补强项，含市场价值评估 |
| 🎯 推荐职业方向 | 2-4 个匹配方向，含匹配理由、薪资参考、晋升路径、进入难度 |
| 📅 行动计划 | 30 天 / 90 天 / 180 天 具体可执行步骤 |
| 💡 副业路径 | 适合你的副业方向（如有相关背景） |
| ⚠️ 诚实的挑战 | 你将面临的真实障碍及应对策略 |
| 📚 资源推荐 | 针对你目标方向的课程、平台、社区 |

## 🚀 快速开始

### 方式一：通过 ClawhHub 安装（推荐）

在 ClawhHub 搜索 `career-planner`，点击安装即可在 Claude.ai 中使用。

### 方式二：手动安装

1. 下载本仓库的 `career-planner.skill` 文件
2. 在 Claude.ai 进入 **Settings → Skills**
3. 点击 **Import Skill**，上传 `.skill` 文件
4. 安装完成后，直接在对话中说"帮我做职业规划"即可触发

## 🗂️ 文件结构

```
career-planner/
├── SKILL.md                        # 主控文件：对话流程、规则、阶段控制
└── references/
    ├── report-template.md          # 报告模板：8 大板块结构与格式规范
    ├── skill-to-role-map.md        # 技能 → 岗位映射矩阵（10 大技能集群）
    └── role-database.md            # 岗位详情库：薪资、技能要求、晋升路径
```

### 设计理念

Skill 采用**渐进式加载**架构：

- `SKILL.md` 主文件常驻上下文，控制对话逻辑（< 500 行）
- 三个 `references/` 文件按需读取，避免 token 浪费
- 联网搜索仅在必要时触发（薪资数据、新兴岗位、细分城市）

## 🔄 对话流程

```
Phase 1  暖场开场       建立信任，设定预期
   ↓
Phase 2  结构化访谈     6 大主题，逐步深入（每个主题 1-3 个问题）
   ↓
Phase 3  综合确认       复述理解，确认无误，填补信息缺口
   ↓
Phase 4  联网调研       查询薪资数据、岗位需求趋势（按需）
   ↓
Phase 5  生成报告       输出个性化规划 → 保存为可下载 .md 文件
```

每个阶段有明确的**推进条件**，不会在信息不足时仓促生成报告。

## 🎯 适合人群

- 🎓 **应届毕业生** — 不知道第一份工作从哪里开始
- 🔄 **职场转型期** — 想换方向但不确定去哪
- 😔 **对现状迷茫** — 做了几年感觉没有成长
- 💼 **想探索副业** — 有技能想变现，不知道从何入手
- 📈 **寻求突破** — 想在现有基础上找到更高价值的方向

---

## 🛠️ 特殊场景处理

| 场景 | 处理策略 |
|------|---------|
| 用户回答模糊 | 提供 A/B/C 多选项引导，不强迫精确 |
| 零经验用户 | 聚焦可迁移技能、学习项目、个人兴趣 |
| 资深职场人 | 侧重战略定位、领导力、影响力话题 |
| 处于危机中（裁员等）| 先共情，再规划；报告增加"快速入门路径" |
| 副业导向用户 | 重点挖掘变现路径，报告增加专属副业板块 |
