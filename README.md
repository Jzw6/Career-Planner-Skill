# Career-Planner-Skill
这是一个专业的 AI 职业规划顾问 Skill。它不会直接给你一堆泛泛的建议，而是像真正的顾问一样，通过自然对话一步步了解你：你的背景与经历、核心技能与优势、兴趣热情与价值观、副业资产与隐藏潜力、生活约束与收入目标、时间紧迫性与职业恐惧。 掌握足够信息后，它会联网查询真实薪资数据和岗位需求，最终为你生成一份可下载的个性化职业规划报告，包含：个人画像分析、技能盘点表、2-4 个推荐职业方向（含匹配理由、薪资参考、晋升路径）、30/90/180 天行动计划、副业可能性、以及诚实的挑战分析。 适合：职场转型期、应届毕业生、想探索副业、对现状迷茫、或任何想认真规划职业的人。
